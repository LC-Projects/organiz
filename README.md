# Organiz App

![Jonathan M. & Lucky M. - Organiz App](https://s6.gifyu.com/images/S6b8I.gif "Demo")

By Jonathan M. & Lucky M.

## What can you do?

### Authentication

- Signup (with mail verification)
- Signin

### Usage

- Boards
  - Add
  - Edit
  - Remove
- In each Board, Tasks
  - Add
  - Edit
  - Remove
- Profile
  - Edit Email
  - Light / Dark Mode switcher
  - Delete Account
  - Logout

## Installation

```CMD
npm install
```

Or

```CMD
yarn
```

## Launch

```CMD
npm start
```

Or

```CMD
yarn start
```
